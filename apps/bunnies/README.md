# PixiVue-Lab

A Turborepo monorepo with Vue 3 and PixiJS v8.

## Structure

- `apps/web`: Vue 3 + Vite + PixiJS Application.
- `packages/`: Shared packages (to be added).

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `bun install`.
3.  Run the development server: `bun dev`.

## Tech Stack

- [Turborepo](https://turbo.build/)
- [Vue 3](https://vuejs.org/)
- [PixiJS v8](https://pixijs.com/)
- [Bun](https://bun.sh/)
- [Vite](https://vite.dev/)
- [Biome](https://biomejs.dev/)
